## 🔹 기본 Key 조작

|목적|명령어|설명|
|---|---|---|
|Key 확인|`KEYS pattern`|예: `KEYS *user*`|
|Key 존재 여부|`EXISTS key`|존재하면 1|
|Key 삭제|`DEL key1 key2`|여러 개 가능|
|TTL 확인|`TTL key`|남은 만료시간(초)|
|만료시간 설정|`EXPIRE key 60`|60초 뒤 삭제|
|즉시 만료 해제|`PERSIST key`|TTL 제거|

---

## 🔹 String

|목적|명령어|설명|
|---|---|---|
|값 저장|`SET key value`|기본 저장|
|값 가져오기|`GET key`||
|증가|`INCR key`|숫자형 증가|
|감소|`DECR key`|숫자형 감소|
|부분 문자열|`GETRANGE key start end`|부분 추출|
|여러 개 set/get|`MSET k1 v1 k2 v2`, `MGET k1 k2`||

## 🔹 Hash (Dictionary 형태)

|목적|명령어|설명|
|---|---|---|
|필드 저장|`HSET user:1 name "Tom" age 30`||
|필드 조회|`HGET user:1 name`||
|전체 조회|`HGETALL user:1`||
|특정 필드 여러개|`HMGET user:1 name age`||
|필드 존재 여부|`HEXISTS user:1 name`||
|필드 삭제|`HDEL user:1 age`||

## 🔹 List (Queue / Stack)

|목적|명령어|설명|
|---|---|---|
|왼쪽 삽입|`LPUSH list a b c`||
|오른쪽 삽입|`RPUSH list x y z`||
|왼쪽 꺼내기|`LPOP list`||
|오른쪽 꺼내기|`RPOP list`||
|범위 조회|`LRANGE list 0 -1`|전체 조회|
|길이|`LLEN list`|리스트 길이|

## 🔹 Set (중복 없는 집합)

|목적|명령어|설명|
|---|---|---|
|추가|`SADD set1 a b c`||
|삭제|`SREM set1 a`||
|포함 여부|`SISMEMBER set1 a`||
|전체 조회|`SMEMBERS set1`||
|교집합|`SINTER s1 s2`||
|합집합|`SUNION s1 s2`||

## 🔹 Sorted Set (랭킹/점수)

|목적|명령어|설명|
|---|---|---|
|추가|`ZADD rank 100 "tom"`||
|점수 조회|`ZSCORE rank tom`||
|오름차순 조회|`ZRANGE rank 0 -1 WITHSCORES`||
|내림차순 조회|`ZREVRANGE rank 0 -1 WITHSCORES`||
|점수 변경|`ZINCRBY rank 10 tom`||
|순위 조회|`ZRANK rank tom`||
