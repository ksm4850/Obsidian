# 배달 시간 최소화

난이도: ★★★☆☆ (중급)
관련: [[이분 탐색]]

## 문제

배달 기사 K명이 N개의 택배를 배달해야 한다. 각 택배의 배달 소요 시간이 배열 `times`로 주어진다.

**규칙:**
- 택배는 **순서대로** 연속된 묶음으로 나눠야 한다 (섞을 수 없음)
- 각 기사에게 **연속된 택배 묶음**을 배정한다
- 모든 기사가 **동시에** 출발한다
- 전체 배달 완료 시간 = 가장 오래 걸리는 기사의 시간

**목표:** 전체 배달 완료 시간의 **최솟값**을 구하라.

## 입력

```
times = [7, 2, 5, 10, 8]
K = 2
```

## 출력

```
18
```

## 설명

```
기사1: [7, 2, 5]     → 14
기사2: [10, 8]        → 18
전체: max(14, 18) = 18  ← 최적

다른 분배:
기사1: [7, 2, 5, 10]  → 24
기사2: [8]             → 8
전체: max(24, 8) = 24  ← 더 나쁨

기사1: [7]             → 7
기사2: [2, 5, 10, 8]   → 25
전체: max(7, 25) = 25  ← 더 나쁨
```

## 추가 테스트 케이스

```python
# 테스트 1
times = [7, 2, 5, 10, 8]
K = 2
# 답: 18

# 테스트 2
times = [1, 2, 3, 4, 5]
K = 3
# 답: 6  ([1,2,3], [4], [5])

# 테스트 3
times = [5, 5, 5, 5]
K = 2
# 답: 10  ([5,5], [5,5])

# 테스트 4
times = [10, 1, 1, 1, 1]
K = 2
# 답: 10  ([10], [1,1,1,1])

# 테스트 5 (기사 수 = 택배 수)
times = [3, 7, 2]
K = 3
# 답: 7  각자 1개씩
```

## 제약

- 1 ≤ K ≤ N ≤ 100,000
- 1 ≤ times[i] ≤ 10,000
