# 분할 정복

## 개요

문제를 작은 부분으로 나누어 풀고 결합. 마스터 정리로 복잡도 분석: `T(n) = a·T(n/b) + f(n)`.

대표: 머지소트 O(N log N), 퀵소트 평균 O(N log N), Strassen 행렬 곱 O(N^2.81).

## 머지소트

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr[:]
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)

def merge(a, b):
    i = j = 0
    res = []
    while i < len(a) and j < len(b):
        if a[i] <= b[j]:
            res.append(a[i]); i += 1
        else:
            res.append(b[j]); j += 1
    res.extend(a[i:]); res.extend(b[j:])
    return res
```

안정 정렬이라는 점이 실전 차별점. 하지만 Python은 `list.sort()`가 Timsort라 보통 그걸 쓴다 — **이 알고리즘 자체를 써야 할 땐 보통 "부산물"이 필요한 경우**.

## 역원 카운트 (머지소트 부산물)

합치며 "우측이 좌측보다 먼저 오면 역원"이라는 걸 세면서 정렬.

```python
def count_inversions(arr):
    def sort(a):
        if len(a) <= 1: return a, 0
        mid = len(a) // 2
        left, li = sort(a[:mid])
        right, ri = sort(a[mid:])
        merged = []
        i = j = cnt = 0
        while i < len(left) and j < len(right):
            if left[i] <= right[j]:
                merged.append(left[i]); i += 1
            else:
                merged.append(right[j]); j += 1
                cnt += len(left) - i
        merged.extend(left[i:]); merged.extend(right[j:])
        return merged, li + ri + cnt
    _, total = sort(arr)
    return total
```

[[Fenwick Tree]] 버전보다 상수 조금 느리지만 코드 단순.

## 퀵셀렉트 — K번째 원소 O(N) 평균

피벗 분할 후 **한쪽만 재귀**. 평균 O(N), 최악 O(N²) (무작위 피벗이면 기대 O(N)).

```python
import random

def quick_select(arr, k):
    """ 0-indexed: k번째로 작은 값 """
    arr = arr[:]
    def partition(l, r, pivot_idx):
        pivot = arr[pivot_idx]
        arr[pivot_idx], arr[r] = arr[r], arr[pivot_idx]
        store = l
        for i in range(l, r):
            if arr[i] < pivot:
                arr[i], arr[store] = arr[store], arr[i]
                store += 1
        arr[store], arr[r] = arr[r], arr[store]
        return store

    l, r = 0, len(arr) - 1
    while l <= r:
        pivot_idx = random.randint(l, r)
        new_idx = partition(l, r, pivot_idx)
        if new_idx == k:
            return arr[new_idx]
        elif new_idx < k:
            l = new_idx + 1
        else:
            r = new_idx - 1
```

[[Heap]]의 O(N log K) 대비 평균 O(N). 랜덤 피벗 필수.

## 가장 가까운 두 점 — O(N log N)

x 정렬 후 중간선으로 분할. 좌·우 재귀 후 **중간 띠 내에서 y 정렬**, 각 점당 다음 7개만 비교.

상세 구현은 길지만 교과서적. 경쟁대회에선 "N ≤ 10⁵, 브루트포스 불가"일 때 등장.

## CDQ 분할 정복

오프라인 3차원 문제를 시간·공간·값 3축으로 분할 정복해 처리. 고급 기법.

## 함정

- 머지소트에서 슬라이싱 `a[:mid]`는 복사라 메모리 두 배. 제자리 정렬 버전은 더 복잡.
- 퀵셀렉트의 최악 O(N²) — 고정 피벗이면 정렬된 입력에서 터짐. 반드시 랜덤 또는 median-of-medians.

## 관련

[[Fenwick Tree]]로도 역원 카운트 가능. K번째 원소는 [[Heap]]. 병합 개념은 [[Two Pointers]]의 merge와 동일.

## 실행 예시

```python
import random

# --- 머지소트 ---
def merge_sort(arr):
    if len(arr) <= 1: return arr[:]
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    i = j = 0; res = []
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            res.append(left[i]); i += 1
        else:
            res.append(right[j]); j += 1
    res.extend(left[i:]); res.extend(right[j:])
    return res

print("머지소트:", merge_sort([5, 2, 8, 1, 9, 3]))
# [1, 2, 3, 5, 8, 9]

# --- 역원 카운트 ---

        right, ri = sort(a[mid:])
        merged = []; i = j = cnt = 0
        while i < len(left) and j < len(right):
            if left[i] <= right[j]:
                merged.append(left[i]); i += 1
            else:
                merged.append(right[j]); j += 1
                cnt += len(left) - i
        merged.extend(left[i:]); merged.extend(right[j:])

print("역원 수 [3,1,2,4]:", count_inversions([3, 1, 2, 4]))   # 2
print("역원 수 [5,4,3,2,1]:", count_inversions([5, 4, 3, 2, 1]))   # 10

    def partition(l, r, pivot_idx):
        pivot = arr[pivot_idx]
        arr[pivot_idx], arr[r] = arr[r], arr[pivot_idx]
        store = l
        for i in range(l, r):

                store += 1
        arr[store], arr[r] = arr[r], arr[store]
        return store
    l, r = 0, len(arr) - 1
    while l <= r:
        pivot_idx = random.randint(l, r)
        new_idx = partition(l, r, pivot_idx)

            l = new_idx + 1
        else:
            r = new_idx - 1

arr = [7, 2, 9, 4, 1, 8, 3, 5, 6]
print("3번째로 작은 값:", quick_select(arr, 2))   # 3
print("5번째로 작은 값:", quick_select(arr, 4))   # 5
print("중앙값:", quick_select(arr, len(arr)//2))  # 5
```
