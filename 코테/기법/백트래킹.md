# 백트래킹

## 개요

모든 가능한 상태를 DFS로 탐색하되, **유망하지 않으면 가지치기**. 탐색 트리를 가지치기로 축소해 지수 시간의 상수/지수를 줄임.

N-Queen, 스도쿠, 순열/조합 생성, 부분집합 합, 제약 만족 문제(CSP).

## 기본 패턴

```python
def backtrack(state):
    if is_solution(state):
        record(state)
        return
    for choice in choices(state):
        if promising(state, choice):
            make(state, choice)
            backtrack(state)
            undo(state, choice)
```

핵심:
1. 상태 복제 대신 **in-place + undo**로 메모리 절약
2. `promising` (가지치기)을 strong하게 짜면 실제 수행 시간 크게 감소

## N-Queen

각 행에 퀸 하나씩, 같은 열·대각선 금지.

```python
def n_queens(n):
    cols = set()
    diag1 = set()   # r - c
    diag2 = set()   # r + c
    board = []
    answers = []

    def bt(r):
        if r == n:
            answers.append(board[:])
            return
        for c in range(n):
            if c in cols or (r - c) in diag1 or (r + c) in diag2:
                continue
            cols.add(c); diag1.add(r - c); diag2.add(r + c)
            board.append(c)
            bt(r + 1)
            board.pop()
            cols.remove(c); diag1.remove(r - c); diag2.remove(r + c)

    bt(0)
    return answers
```

## 순열 / 조합 생성

```python
def permutations(arr):
    n = len(arr)
    used = [False] * n
    path = []
    res = []
    def bt():
        if len(path) == n:
            res.append(path[:])
            return
        for i in range(n):
            if used[i]: continue
            used[i] = True; path.append(arr[i])
            bt()
            path.pop(); used[i] = False
    bt()
    return res

def combinations(arr, k):
    n = len(arr)
    path = []
    res = []
    def bt(start):
        if len(path) == k:
            res.append(path[:]); return
        for i in range(start, n):
            path.append(arr[i])
            bt(i + 1)
            path.pop()
    bt(0)
    return res
```

조합은 `start`로 중복 방지. `itertools.permutations`, `itertools.combinations`도 충분히 빠름.

## 가지치기 전략

- **하한 가지치기**: 남은 선택으로 최선을 다해도 현재 최적보다 나쁘면 중단
- **상한 가지치기**: 불가능 조건 조기 탐지 (예: 스도쿠에서 한 셀에 넣을 수 있는 숫자가 0)
- **대칭 제거**: N-Queen에서 첫 행을 왼쪽 절반만 시도, 나머지는 미러로 곱하기
- **제약 전파**: 선택 시 연관된 다른 셀의 후보 축소 (스도쿠)

## 부분집합 합 — 가지치기 예

```python
def subset_sum_bt(nums, target):
    nums.sort(reverse=True)   # 큰 것부터 시도하면 빨리 가지치기
    n = len(nums)
    suffix_sum = [0] * (n + 1)
    for i in range(n - 1, -1, -1):
        suffix_sum[i] = suffix_sum[i+1] + nums[i]

    found = [False]

    def bt(i, cur):
        if found[0]: return
        if cur == target:
            found[0] = True; return
        if i == n: return
        if cur + suffix_sum[i] < target: return   # 남은 전부 더해도 부족
        if cur + nums[i] <= target:
            bt(i + 1, cur + nums[i])
        bt(i + 1, cur)

    bt(0, 0)
    return found[0]
```

## 백트래킹 vs [[Bitmask DP]]

같은 문제를 둘 다로 풀 수 있을 때가 많음. 차이:

- **백트래킹**: 중복 상태 저장 X → 공간 O(재귀 깊이), 시간 최악 지수
- **Bitmask DP**: 상태(2^N)만큼 메모리 사용, 시간 O(2^N × N)

N ≤ 20이면서 중복 부분 문제가 있으면 DP, 그 이상이거나 고유 해집합이 필요하면 백트래킹.

## 함정

- `path[:]`(복사)을 안 하고 `path`를 저장하면 모든 결과가 같은 참조.
- 재귀 깊이 체크. N = 50 정도는 안전하지만 더 깊으면 `sys.setrecursionlimit`.
- "대칭" 해를 중복으로 세지 않도록 주의.

## 관련

[[BFS DFS 응용]]의 DFS와 원리 같음, 대신 **탐색 목표가 해집합**이라는 차이. 탐색 공간을 메모이즈하면 [[Bitmask DP]]·[[DP 패턴]].

def n_queens(n):
    cols = set(); diag1 = set(); diag2 = set()
    board = []; answers = []
    def bt(r):
        if r == n:
            answers.append(board[:]); return
        for c in range(n):
            if c in cols or (r - c) in diag1 or (r + c) in diag2:

print("4-Queen 해 개수:", len(ans))    # 2
print("해:", ans)                      # [[1,3,0,2], [2,0,3,1]]

def subset_sum_bt(nums, target):
    nums.sort(reverse=True)
    n = len(nums)
    suffix_sum = [0] * (n + 1)
    for i in range(n - 1, -1, -1):
        suffix_sum[i] = suffix_sum[i+1] + nums[i]
    found = [False]
    def bt(i, cur):
        if found[0]: return
        if cur == target:
            found[0] = True; return
        if i == n: return
        if cur + suffix_sum[i] < target: return
        if cur + nums[i] <= target:
            bt(i + 1, cur + nums[i])
        bt(i + 1, cur)
    bt(0, 0)
    return found[0]

print("부분합 15:", subset_sum_bt([3, 5, 7, 10, 12], 15))   # True (3+12 or 5+10)
print("부분합 14:", subset_sum_bt([3, 5, 7, 10, 12], 14))   # False
