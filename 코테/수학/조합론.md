# 조합론

## 개요

코테에서 자주 쓰는 조합·순열·카탈란 수. 값이 커서 mod p 하에 [[모듈러 역원]] 전처리와 함께 다루는 게 표준.

## 파스칼 삼각형 — O(N²) 전처리

nCr을 `dp[n][r]`로 누적. N, R ≤ 1000 정도일 때.

```python
def pascal(N, MOD):
    C = [[0]*(N+1) for _ in range(N+1)]
    for i in range(N+1):
        C[i][0] = 1
        for j in range(1, i+1):
            C[i][j] = (C[i-1][j-1] + C[i-1][j]) % MOD
    return C
```

## 팩토리얼 + 역원 — O(N) 전처리, 쿼리 O(1)

N ≤ 10⁶일 때 표준.

```python
MOD = 10**9 + 7

def precompute(N):
    fact = [1] * (N + 1)
    for i in range(1, N + 1):
        fact[i] = fact[i-1] * i % MOD
    inv = [1] * (N + 1)
    inv[N] = pow(fact[N], MOD - 2, MOD)
    for i in range(N - 1, -1, -1):
        inv[i] = inv[i+1] * (i+1) % MOD
    return fact, inv

def nCr(n, r, fact, inv):
    if r < 0 or r > n: return 0
    return fact[n] * inv[r] % MOD * inv[n-r] % MOD
```

상세 전처리 유도는 [[모듈러 역원]] 참조.

## 중복 조합 H(n, r) = C(n+r-1, r)

"n종류에서 중복 허용 r개 뽑기". "공 r개를 구분 없는 n개의 상자에 넣기".

```python
def nHr(n, r, fact, inv):
    return nCr(n + r - 1, r, fact, inv)
```

## 카탈란 수

```
C_n = C(2n, n) / (n + 1) = C(2n, n) - C(2n, n+1)
```

괄호 매칭, 이진트리 개수, 모노톤 격자 경로 등 빈출.

```python
def catalan(n, fact, inv):
    return nCr(2*n, n, fact, inv) * pow(n + 1, MOD - 2, MOD) % MOD
```

처음 몇 값: 1, 1, 2, 5, 14, 42, 132, 429, 1430, ...

## 스털링 수 (2종)

n개를 k개의 **비어있지 않은 부분집합**으로 나누는 경우의 수.

점화식: `S(n, k) = k·S(n-1, k) + S(n-1, k-1)`.

```python
def stirling2(N, K, MOD=10**9+7):
    S = [[0]*(K+1) for _ in range(N+1)]
    S[0][0] = 1
    for n in range(1, N+1):
        for k in range(1, K+1):
            S[n][k] = (k * S[n-1][k] + S[n-1][k-1]) % MOD
    return S
```

## 벨 수 (Bell)

n개 원소 집합의 **모든 분할 수** = Σ S(n, k).

## 포함-배제 원리

|A∪B∪C| = |A|+|B|+|C| − |A∩B|−|A∩C|−|B∩C| + |A∩B∩C|.

**서로소 조건 부분집합 개수**류 문제의 핵심.

```python
def derangement(n, MOD=10**9+7):
    """교란순열 D(n) = n! · Σ (-1)^k / k!"""
    fact, inv = precompute(n)
    ans = 0
    for k in range(n + 1):
        sign = 1 if k % 2 == 0 else MOD - 1
        ans = (ans + sign * inv[k]) % MOD
    return ans * fact[n] % MOD
```

## 격자 경로 (오른/위만 이동)

(0,0) → (n, m)은 `C(n+m, n)`. 대각선 지나면 안 되는 제한은 카탈란 수 변형.

## 함정

- 음수 인자·범위 밖 `nCr`은 0으로 처리.
- MOD가 소수가 아니면 역원 계산 실패 → Lucas 정리 또는 CRT.

## 관련

구현은 [[모듈러 역원]]에 의존. 큰 수 mod 소수 계산은 [[행렬 거듭제곱]]과 다른 축의 전처리 기법.

## 실행 예시

```python
MOD = 10**9 + 7

# --- 파스칼 삼각형 ---
def pascal(N, MOD):
    C = [[0]*(N+1) for _ in range(N+1)]
    for i in range(N+1):
        C[i][0] = 1
        for j in range(1, i+1):
            C[i][j] = (C[i-1][j-1] + C[i-1][j]) % MOD
    return C

C = pascal(6, MOD)
print("파스칼 삼각형:")
for i in range(7):
    print(C[i][:i+1])

# --- 팩토리얼 + 역원 전처리 ---
def precompute(N):
    fact = [1] * (N + 1)
    for i in range(1, N + 1):
        fact[i] = fact[i-1] * i % MOD
    inv = [1] * (N + 1)
    inv[N] = pow(fact[N], MOD - 2, MOD)
    for i in range(N - 1, -1, -1):
        inv[i] = inv[i+1] * (i+1) % MOD
    return fact, inv

def nCr(n, r, fact, inv):
    if r < 0 or r > n: return 0
    return fact[n] * inv[r] % MOD * inv[n-r] % MOD

fact, inv = precompute(100)
print("C(100, 50):", nCr(100, 50, fact, inv))
print("C(10, 3):", nCr(10, 3, fact, inv))   # 120

# --- 중복 조합 ---
def nHr(n, r, fact, inv):
    return nCr(n + r - 1, r, fact, inv)

print("H(3, 2):", nHr(3, 2, fact, inv))   # 6 (3종류에서 중복 허용 2개)

# --- 카탈란 수 ---
def catalan(n, fact, inv):
    return nCr(2*n, n, fact, inv) * pow(n + 1, MOD - 2, MOD) % MOD

print("카탈란 0~10:", [catalan(i, fact, inv) for i in range(11)])
# [1, 1, 2, 5, 14, 42, 132, 429, 1430, 4862, 16796]

# --- 스털링 수 (2종) ---
def stirling2(N, K, MOD=10**9+7):
    S = [[0]*(K+1) for _ in range(N+1)]
    S[0][0] = 1
    for n in range(1, N+1):
        for k in range(1, K+1):
            S[n][k] = (k * S[n-1][k] + S[n-1][k-1]) % MOD
    return S

S = stirling2(5, 5)
print("S(5, 2):", S[5][2])   # 15
print("S(5, 3):", S[5][3])   # 25

# --- 교란순열 ---
def derangement(n):
    fact, inv = precompute(n)
    ans = 0
    for k in range(n + 1):
        sign = 1 if k % 2 == 0 else MOD - 1
        ans = (ans + sign * inv[k]) % MOD
    return ans * fact[n] % MOD

print("D(4):", derangement(4))   # 9
print("D(5):", derangement(5))   # 44
```
