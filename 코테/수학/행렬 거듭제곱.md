# 행렬 거듭제곱 (Matrix Exponentiation)

## 개요

`M^k`를 **반복제곱법**으로 O(d³ · log k)에. d는 행렬 크기. 선형 점화식(피보나치 등)을 O(log N)으로 계산.

## 핵심: 거듭제곱

```python
def mat_mul(A, B, mod=None):
    n, m, p = len(A), len(A[0]), len(B[0])
    C = [[0]*p for _ in range(n)]
    for i in range(n):
        Ai = A[i]
        Ci = C[i]
        for k in range(m):
            a = Ai[k]
            if a == 0: continue
            Bk = B[k]
            for j in range(p):
                Ci[j] += a * Bk[j]
                if mod: Ci[j] %= mod
    return C

def mat_pow(M, k, mod=None):
    n = len(M)
    # 단위 행렬
    result = [[1 if i == j else 0 for j in range(n)] for i in range(n)]
    while k > 0:
        if k & 1:
            result = mat_mul(result, M, mod)
        M = mat_mul(M, M, mod)
        k >>= 1
    return result
```

루프 안에서 `a == 0` 스킵은 Python에서 꽤 큰 상수 절감.

## 피보나치 O(log N)

점화식 `[F(n+1), F(n)] = [[1,1],[1,0]] · [F(n), F(n-1)]`.

```python
def fib(n, mod=10**9+7):
    if n == 0: return 0
    M = [[1, 1], [1, 0]]
    R = mat_pow(M, n, mod)
    return R[0][1]      # F(n)
```

n = 10¹⁸도 1초 내 가능.

## 선형 점화식 일반화

`a(n) = c1·a(n-1) + c2·a(n-2) + ... + ck·a(n-k)` 를 k차 행렬로 표현.

```python
def linear_recurrence(coefs, init, n, mod):
    """
    coefs = [c1, c2, ..., ck]
    init = [a(0), a(1), ..., a(k-1)]
    """
    k = len(coefs)
    if n < k:
        return init[n] % mod
    # 전이 행렬
    M = [[0]*k for _ in range(k)]
    for i in range(k):
        M[0][i] = coefs[i]
    for i in range(1, k):
        M[i][i-1] = 1
    R = mat_pow(M, n - k + 1, mod)
    # 결과: R · [a(k-1), ..., a(0)]
    vec = init[::-1]
    return sum(R[0][j] * vec[j] for j in range(k)) % mod
```

## 응용 1: 그래프에서 길이 K 경로 수

인접행렬 A에 대해 `A^K [i][j]` = i→j 길이 K 경로 수.

```python
def count_paths(adj, K, mod):
    return mat_pow(adj, K, mod)
```

## 응용 2: DP 전이가 행렬인 문제

상태 수 d가 작고(≤ 100) 시간 축이 매우 큰(10⁹) DP는 전이를 행렬로 표현해 log로 줄임.

전형 예: "길이 N인 문자열 중 부분문자열 금지 조건 만족" — 상태가 금지 부분 접미사 길이이면 d가 작음.

## 응용 3: 기하 변환

회전·평행이동을 4×4 동차좌표 행렬 곱. 반복 변환의 합성을 거듭제곱으로.

## 함정

- `mod`가 있을 때 **곱셈마다** `% mod`. 생략하면 수가 폭발해 연산이 느려짐.
- d³ 곱셈은 d=100이면 10⁶, 로그 60회 = 6×10⁷ → Python tight. PyPy 권장 경계.
- 단위 행렬 초기화 잊지 않기.

## 관련

[[조합론]]의 점화식도 일부는 행렬화 가능. 카탈란 수 단순 점화식은 이 방식 부적합(비선형).

## 실행 예시

```python
def mat_mul(A, B, mod=None):
    n, m, p = len(A), len(A[0]), len(B[0])
    C = [[0]*p for _ in range(n)]
    for i in range(n):
        Ai = A[i]; Ci = C[i]
        for k in range(m):
            a = Ai[k]
            if a == 0: continue
            Bk = B[k]
            for j in range(p):
                Ci[j] += a * Bk[j]
                if mod: Ci[j] %= mod
    return C

def mat_pow(M, k, mod=None):
    n = len(M)
    result = [[1 if i == j else 0 for j in range(n)] for i in range(n)]
    while k > 0:
        if k & 1:
            result = mat_mul(result, M, mod)
        M = mat_mul(M, M, mod)
        k >>= 1
    return result

# --- 피보나치 ---
def fib(n, mod=10**9+7):
    if n == 0: return 0
    M = [[1, 1], [1, 0]]
    R = mat_pow(M, n, mod)
    return R[0][1]

print("fib(10):", fib(10))                  # 55
print("fib(50):", fib(50))                  # 12586269025
print("fib(10^9) mod:", fib(10**9))         # 큰 수도 빠름

# --- 인접행렬 길이 K 경로 ---
def count_paths(adj, K, mod):
    return mat_pow(adj, K, mod)

# 그래프: 0-1, 1-2, 2-0 (삼각형)
adj = [
    [0, 1, 1],
    [1, 0, 1],
    [1, 1, 0],
]
P3 = count_paths(adj, 3, 10**9+7)
print("길이 3 경로 (0->0):", P3[0][0])   # 2
print("길이 3 경로 (0->1):", P3[0][1])   # 3

# --- 선형 점화식 일반 ---
def linear_recurrence(coefs, init, n, mod):
    k = len(coefs)
    if n < k:
        return init[n] % mod
    M = [[0]*k for _ in range(k)]
    for i in range(k):
        M[0][i] = coefs[i]
    for i in range(1, k):
        M[i][i-1] = 1
    R = mat_pow(M, n - k + 1, mod)
    vec = init[::-1]
    return sum(R[0][j] * vec[j] for j in range(k)) % mod

# a(n) = a(n-1) + a(n-2) (피보나치)
print("fib(10) via general:", linear_recurrence([1, 1], [0, 1], 10, 10**9+7))   # 55
# a(n) = 2a(n-1) + a(n-2), init [1, 1] -> 펠 수와 유사
print("a(10):", linear_recurrence([2, 1], [1, 1], 10, 10**9+7))
```
