# 확장 유클리드 (Extended Euclidean)

## 개요

`gcd(a, b)`와 함께 **베주 항등식** `a·x + b·y = gcd(a, b)`을 만족하는 `(x, y)`를 반환. 디오판토스 방정식, [[모듈러 역원]], CRT의 핵심.

## GCD

```python
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

# Python 표준 라이브러리
from math import gcd
```

## 확장 유클리드

재귀:

```python
def extgcd(a, b):
    """ return (g, x, y): a*x + b*y = g = gcd(a, b) """
    if b == 0:
        return a, 1, 0
    g, x1, y1 = extgcd(b, a % b)
    return g, y1, x1 - (a // b) * y1
```

반복 버전(스택 안전):

```python
def extgcd_iter(a, b):
    old_r, r = a, b
    old_s, s = 1, 0
    old_t, t = 0, 1
    while r:
        q = old_r // r
        old_r, r = r, old_r - q * r
        old_s, s = s, old_s - q * s
        old_t, t = t, old_t - q * t
    return old_r, old_s, old_t
```

## 모듈러 역원 (gcd(a, m) = 1)

`a·x ≡ 1 (mod m)`의 해. 확장 유클리드에서 x가 역원.

```python
def mod_inv(a, m):
    g, x, _ = extgcd(a, m)
    if g != 1:
        return None   # 역원 없음
    return x % m

# Python 3.8+
inv = pow(a, -1, m)
```

`pow(a, -1, m)`가 제일 간편. 내부적으로 확장 유클리드를 쓴다.

## 선형 디오판토스 방정식

`a·x + b·y = c`의 정수해. `gcd(a, b) | c`여야 해 존재.

```python
def solve_linear(a, b, c):
    """ 하나의 (x0, y0). 일반해: x = x0 + (b/g)t, y = y0 - (a/g)t """
    g, x, y = extgcd(a, b)
    if c % g != 0:
        return None
    factor = c // g
    return x * factor, y * factor
```

## CRT (중국인의 나머지 정리)

`x ≡ r_i (mod m_i)`, i = 1..k. `m_i`가 서로소일 때 0 ≤ x < M = ∏m_i 유일.

```python
def crt(rems, mods):
    from math import prod
    M = prod(mods)
    x = 0
    for r, m in zip(rems, mods):
        Mi = M // m
        x = (x + r * Mi * pow(Mi, -1, m)) % M
    return x
```

일반화(서로소 아닌 경우): 둘씩 합쳐가며 `gcd(m1, m2) | (r2 - r1)` 체크.

## 함정

- `x`가 음수일 수 있음 → 역원 결과는 `x % m`.
- 재귀 버전은 큰 수에 대해 스택 깊이 문제 없음(로그 스케일). 하지만 정수 자체가 매우 크면 Python은 괜찮고 다른 언어는 오버플로 주의.

## 관련

[[모듈러 역원]]의 계산 수단. [[에라토스테네스]]와 함께 정수론 기본. [[조합론]]의 nCr mod p에서 역원 필요.

## 실행 예시

```python
from math import gcd, prod

# --- GCD ---
print("gcd(24, 36):", gcd(24, 36))   # 12
print("gcd(17, 5):", gcd(17, 5))     # 1 (서로소)

# --- 확장 유클리드 ---
def extgcd(a, b):
    if b == 0:
        return a, 1, 0
    g, x1, y1 = extgcd(b, a % b)
    return g, y1, x1 - (a // b) * y1

g, x, y = extgcd(30, 18)
print(f"30·{x} + 18·{y} = {g}")    # 30·(-1) + 18·2 = 12 -> -30+36=6? 다시 계산: gcd=6
# 실제: gcd(30,18)=6, 30·-1 + 18·2 = 6  ✓

# --- 모듈러 역원 ---
def mod_inv(a, m):
    g, x, _ = extgcd(a, m)
    if g != 1: return None
    return x % m

print("inv(3, 11):", mod_inv(3, 11))   # 4 (3*4=12 ≡ 1 mod 11)
print("inv(6, 9):", mod_inv(6, 9))     # None (gcd=3)
print("pow(3, -1, 11):", pow(3, -1, 11))  # 4 (Python 3.8+)

# --- 선형 디오판토스 ax + by = c ---
def solve_linear(a, b, c):
    g, x, y = extgcd(a, b)
    if c % g != 0: return None
    factor = c // g
    return x * factor, y * factor

sol = solve_linear(6, 10, 14)
print(f"6x + 10y = 14 해: x={sol[0]}, y={sol[1]}")
# 예: x=-7, y=7 -> 6*(-7) + 10*7 = -42+70 = 28? 확인: gcd(6,10)=2, factor=7
# extgcd(6,10): 6*2 + 10*(-1) = 2, *7 = 6*14 + 10*(-7) = 14  ✓

# --- CRT ---
def crt(rems, mods):
    M = prod(mods)
    x = 0
    for r, m in zip(rems, mods):
        Mi = M // m
        x = (x + r * Mi * pow(Mi, -1, m)) % M
    return x

# x ≡ 2 (mod 3), x ≡ 3 (mod 5), x ≡ 2 (mod 7)
print("CRT:", crt([2, 3, 2], [3, 5, 7]))   # 23
```
