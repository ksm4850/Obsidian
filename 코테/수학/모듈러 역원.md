# 모듈러 역원

## 개요

`a·x ≡ 1 (mod m)`을 만족하는 x. **`gcd(a, m) = 1`일 때만 존재**. 세 가지 계산법:

1. **페르마 소정리** (m이 소수): `a^(m-2) mod m`
2. **확장 유클리드** (일반): [[확장 유클리드]] 참조
3. **선형 시간 전처리** (1~N의 역원 전부): 점화식

## 1. 페르마 소정리 (m이 소수)

m이 소수이고 `gcd(a, m) = 1`이면 `a^(m-1) ≡ 1 (mod m)` → `a^(m-2) ≡ a⁻¹`.

```python
def inv_fermat(a, p):
    return pow(a, p - 2, p)

# 또는 Python 3.8+
inv = pow(a, -1, p)
```

`pow(a, b, m)`은 제곱 빠른 거듭제곱 O(log b). 가장 흔히 쓰는 방법.

## 2. 확장 유클리드 — m이 소수 아니거나 일반

```python
def mod_inv(a, m):
    return pow(a, -1, m)   # Python 3.8+ 자동으로 extgcd 사용
```

옛 버전이면 [[확장 유클리드]]의 직접 구현.

## 3. 선형 전처리 (1 ~ N의 역원 모두)

`m`이 소수일 때, 다음 점화식으로 O(N):

```
inv[1] = 1
inv[i] = -(m / i) * inv[m % i] mod m
```

```python
def all_inverses(n, m):
    inv = [0] * (n + 1)
    inv[1] = 1
    for i in range(2, n + 1):
        inv[i] = (-(m // i) * inv[m % i]) % m
    return inv
```

팩토리얼 모듈러 역원, nCr mod p 빠른 전처리에 필수.

## 응용: 팩토리얼 역원 전처리

```python
MOD = 10**9 + 7

def precompute_factorials(N):
    fact = [1] * (N + 1)
    for i in range(1, N + 1):
        fact[i] = fact[i-1] * i % MOD
    inv_fact = [1] * (N + 1)
    inv_fact[N] = pow(fact[N], MOD - 2, MOD)
    for i in range(N - 1, -1, -1):
        inv_fact[i] = inv_fact[i+1] * (i+1) % MOD
    return fact, inv_fact
```

역순 점화식: `inv_fact[i] = inv_fact[i+1] * (i+1)`. 단 한 번의 `pow`로 전처리 O(N).

## nCr mod p

```python
def nCr_mod(n, r, fact, inv_fact, MOD):
    if r < 0 or r > n: return 0
    return fact[n] * inv_fact[r] % MOD * inv_fact[n-r] % MOD
```

## Lucas 정리 (n, r이 매우 크고 p가 작을 때)

`nCr mod p = ∏ (n_i C r_i) mod p`, 여기서 n_i, r_i는 p진 자릿수.

```python
def lucas(n, r, p, fact, inv_fact):
    if r == 0: return 1
    return lucas(n // p, r // p, p, fact, inv_fact) * \
           nCr_mod(n % p, r % p, fact, inv_fact, p) % p
```

## 함정

- **m이 소수가 아니면** 페르마 소정리 쓸 수 없음. `gcd(a, m) ≠ 1`이면 역원 자체가 없음.
- `pow(a, -1, m)`은 Python 3.8+. 낮은 버전은 직접 구현.
- 0의 역원은 없음 → 0으로 나누는 상황 방지.

## 관련

이론적 근거는 [[확장 유클리드]]. 소수 판정/생성은 [[에라토스테네스]]. 실전 응용은 [[조합론]]의 nCr mod p.

## 실행 예시

```python
MOD = 10**9 + 7

# --- 페르마 소정리 ---
def inv_fermat(a, p):
    return pow(a, p - 2, p)

print("inv(3) mod 7:", inv_fermat(3, 7))   # 5 (3*5=15 ≡ 1 mod 7)
print("pow(3, -1, 7):", pow(3, -1, 7))     # 5 (동일)

# --- 1..N 역원 전처리 ---
def all_inverses(n, m):
    inv = [0] * (n + 1)
    inv[1] = 1
    for i in range(2, n + 1):
        inv[i] = (-(m // i) * inv[m % i]) % m
    return inv

inv = all_inverses(10, 11)
print("1..10의 역원 mod 11:", inv[1:])
# [1, 6, 4, 3, 9, 2, 8, 7, 5, 10]

# --- 팩토리얼 + 역원 전처리 ---
def precompute_factorials(N):
    fact = [1] * (N + 1)
    for i in range(1, N + 1):
        fact[i] = fact[i-1] * i % MOD
    inv_fact = [1] * (N + 1)
    inv_fact[N] = pow(fact[N], MOD - 2, MOD)
    for i in range(N - 1, -1, -1):
        inv_fact[i] = inv_fact[i+1] * (i+1) % MOD
    return fact, inv_fact

fact, inv_fact = precompute_factorials(10)
print("10! mod:", fact[10])                   # 3628800
print("10!의 역원 검증:", fact[10] * inv_fact[10] % MOD)   # 1

def nCr_mod(n, r, fact, inv_fact):
    if r < 0 or r > n: return 0
    return fact[n] * inv_fact[r] % MOD * inv_fact[n-r] % MOD

print("C(10, 3):", nCr_mod(10, 3, fact, inv_fact))   # 120
print("C(10, 5):", nCr_mod(10, 5, fact, inv_fact))   # 252
```
