# 슬라이딩 윈도우

## 개요

"연속된 구간"을 움직이면서 상태를 O(1)에 갱신. 고정 크기·가변 크기 두 유형. 전체 O(N).

## 고정 크기 윈도우

윈도우가 한 칸 밀리면 **들어오는 원소 + / 나가는 원소 -**.

```python
def window_sum(arr, k):
    cur = sum(arr[:k])
    res = [cur]
    for i in range(k, len(arr)):
        cur += arr[i] - arr[i-k]
        res.append(cur)
    return res
```

## 가변 크기 — 조건 만족 최소/최대 길이

"최소 길이 부분배열 합 ≥ S"는 [[Two Pointers]] 참고. 이쪽은 "최대 길이 K개 고유 문자 포함" 유형.

```python
from collections import defaultdict

def longest_k_distinct(s, k):
    count = defaultdict(int)
    l = 0
    best = 0
    for r, c in enumerate(s):
        count[c] += 1
        while len(count) > k:
            count[s[l]] -= 1
            if count[s[l]] == 0:
                del count[s[l]]
            l += 1
        best = max(best, r - l + 1)
    return best
```

핵심 구조:
1. r을 하나 확장 → 상태 업데이트
2. 조건이 깨지면 l을 전진시켜 복구
3. 매 스텝 답 갱신

## 패턴: 정확히 K개 고유 → "≤K" - "≤K-1"

"정확히 K개"를 직접 풀기 복잡할 땐 "**최대 K개** 개수 − **최대 K-1개** 개수".

```python
def count_exactly_k_distinct(s, k):
    return count_at_most_k(s, k) - count_at_most_k(s, k-1)
```

카운트 문제에 일반적으로 통하는 trick.

## 윈도우 최댓값/최솟값 — Monotonic Deque

[[Monotonic Stack]] 참조. 덱에 인덱스를 저장, 값이 단조 감소(최댓값) 또는 증가(최솟값)하도록.

```python
from collections import deque

def sliding_max(arr, k):
    dq = deque()
    res = []
    for i, x in enumerate(arr):
        while dq and arr[dq[-1]] <= x:
            dq.pop()
        dq.append(i)
        if dq[0] <= i - k:
            dq.popleft()
        if i >= k - 1:
            res.append(arr[dq[0]])
    return res
```

## 문자열 빈도 매칭

"t의 모든 문자를 포함하는 s의 최소 부분문자열" (LeetCode 76 유형).

```python
from collections import Counter

def min_window(s, t):
    need = Counter(t)
    missing = len(t)
    l = start = end = 0
    for r, c in enumerate(s, 1):
        if need[c] > 0:
            missing -= 1
        need[c] -= 1
        if missing == 0:
            while l < r and need[s[l]] < 0:
                need[s[l]] += 1
                l += 1
            if end == 0 or r - l <= end - start:
                start, end = l, r
            need[s[l]] += 1
            missing += 1
            l += 1
    return s[start:end]
```

## 함정

- 윈도우 축소 조건을 **while**로 쓸 것. if 하나면 한 번만 줄여서 틀림.
- 상태 자료구조: Counter/defaultdict에서 0이 되면 **삭제**해야 "유일 원소 수"가 맞음.
- 투포인터와 경계가 겹침 — 대부분 같은 패턴의 다른 이름.

## 관련

[[Two Pointers]]와 쌍둥이. 윈도우 최대/최소는 [[Monotonic Stack]]의 덱 버전. 고정 해시 구간 매칭은 [[Rabin-Karp]].

## 실행 예시

```python
from collections import deque, defaultdict, Counter

# --- 고정 크기 윈도우 합 ---
def window_sum(arr, k):
    cur = sum(arr[:k])
    res = [cur]
    for i in range(k, len(arr)):
        cur += arr[i] - arr[i-k]
        res.append(cur)
    return res

print("윈도우 합 (k=3):", window_sum([1, 2, 3, 4, 5, 6], 3))   # [6, 9, 12, 15]

# --- 최대 K개 고유 문자 포함 최장 ---
def longest_k_distinct(s, k):
    count = defaultdict(int)
    l = 0; best = 0
    for r, c in enumerate(s):
        count[c] += 1
        while len(count) > k:
            count[s[l]] -= 1
            if count[s[l]] == 0:
                del count[s[l]]
            l += 1
        best = max(best, r - l + 1)
    return best

print("최장 2-distinct:", longest_k_distinct("eceba", 2))   # 3 ("ece")

# --- 슬라이딩 윈도우 최댓값 ---
def sliding_max(arr, k):
    dq = deque()
    res = []
    for i, x in enumerate(arr):
        while dq and arr[dq[-1]] <= x:
            dq.pop()
        dq.append(i)
        if dq[0] <= i - k:
            dq.popleft()
        if i >= k - 1:
            res.append(arr[dq[0]])
    return res

print("sliding max (k=3):", sliding_max([1, 3, -1, -3, 5, 3, 6, 7], 3))
# [3, 3, 5, 5, 6, 7]

# --- 최소 윈도우 부분 문자열 ---
def min_window(s, t):
    need = Counter(t)
    missing = len(t)
    l = start = end = 0
    for r, c in enumerate(s, 1):
        if need[c] > 0:
            missing -= 1
        need[c] -= 1
        if missing == 0:
            while l < r and need[s[l]] < 0:
                need[s[l]] += 1; l += 1
            if end == 0 or r - l <= end - start:
                start, end = l, r
            need[s[l]] += 1
            missing += 1
            l += 1
    return s[start:end]

print("최소 윈도우:", min_window("ADOBECODEBANC", "ABC"))   # "BANC"
```
