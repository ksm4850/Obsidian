# 위상 정렬 (Topological Sort)

## 개요

DAG(방향 비순환 그래프)의 정점을 "모든 간선 u→v에서 u가 v보다 먼저" 나오도록 나열. **O(V + E)**. 작업 순서, 의존성 해결, DAG 최단경로의 기반.

## Kahn's Algorithm (BFS 기반)

진입차수(in-degree) 0인 정점을 큐에 넣고, 꺼낼 때마다 그 정점에서 나가는 간선을 제거. 큐가 비면 끝.

```python
from collections import deque

def topo_sort(n, graph):
    indeg = [0] * n
    for u in range(n):
        for v in graph[u]:
            indeg[v] += 1
    dq = deque(i for i in range(n) if indeg[i] == 0)
    order = []
    while dq:
        u = dq.popleft()
        order.append(u)
        for v in graph[u]:
            indeg[v] -= 1
            if indeg[v] == 0:
                dq.append(v)
    return order if len(order) == n else None  # None = 사이클
```

## DFS 기반

탈출 시점에 스택에 push, 역순으로 출력.

```python
def topo_sort_dfs(n, graph):
    color = [0] * n  # 0 white, 1 gray, 2 black
    order = []
    def dfs(u):
        color[u] = 1
        for v in graph[u]:
            if color[v] == 1:
                return False       # 사이클
            if color[v] == 0 and not dfs(v):
                return False
        color[u] = 2
        order.append(u)
        return True
    for i in range(n):
        if color[i] == 0 and not dfs(i):
            return None
    return order[::-1]
```

## 사전순 최소 위상 정렬

우선순위 큐(min-heap) 사용. [[Heap]] 참조.

```python
import heapq

def lex_topo_sort(n, graph):
    indeg = [0] * n
    for u in range(n):
        for v in graph[u]:
            indeg[v] += 1
    pq = [i for i in range(n) if indeg[i] == 0]
    heapq.heapify(pq)
    order = []
    while pq:
        u = heapq.heappop(pq)
        order.append(u)
        for v in graph[u]:
            indeg[v] -= 1
            if indeg[v] == 0:
                heapq.heappush(pq, v)
    return order
```

## 응용 1: DAG 최장 경로

위상 순서대로 DP.

```python
def longest_path_dag(n, graph, weight):
    order = topo_sort(n, graph)
    dp = [0] * n
    for u in order:
        for v in graph[u]:
            if dp[u] + weight[(u, v)] > dp[v]:
                dp[v] = dp[u] + weight[(u, v)]
    return max(dp)
```

## 응용 2: 작업 스케줄링 / 선수 과목

각 작업에 소요 시간이 있고, 선수관계를 만족해야 할 때 "모든 작업을 끝내는 최소 시간" = 각 정점 종료 시간의 최댓값.

```python
def min_finish(n, graph, cost):
    order = topo_sort(n, graph)
    finish = [cost[i] for i in range(n)]
    for u in order:
        for v in graph[u]:
            if finish[u] + cost[v] > finish[v]:
                finish[v] = finish[u] + cost[v]
    return max(finish)
```

## 응용 3: 유일 위상 정렬?

Kahn에서 **큐 크기가 항상 1 이하**면 유일. 2 이상이 되는 순간 여러 답 존재.

## 함정

- 그래프가 **DAG가 아니면** 위상 정렬 불가 → `len(order) != n`으로 사이클 판정.
- 간선 방향 실수 주의: "A가 B의 선수"라면 A → B.

## 관련

사이클 탐지는 [[BFS DFS 응용]]의 DFS 색칠 버전과 동등. 강결합 컴포넌트는 [[SCC]]. DP 결합 문제는 [[DP 패턴]].

## 실행 예시

```python
from collections import deque

def topo_sort(n, graph):
    indeg = [0] * n
    for u in range(n):
        for v in graph[u]:
            indeg[v] += 1
    dq = deque(i for i in range(n) if indeg[i] == 0)
    order = []
    while dq:
        u = dq.popleft()
        order.append(u)
        for v in graph[u]:
            indeg[v] -= 1
            if indeg[v] == 0:
                dq.append(v)
    return order if len(order) == n else None

# DAG: 0 → 1 → 3, 0 → 2 → 3, 2 → 4
graph = {0: [1, 2], 1: [3], 2: [3, 4], 3: [], 4: []}
print("topo order:", topo_sort(5, graph))   # [0, 1, 2, 3, 4] 등 (여러 해)

# 사이클 포함 → None
graph_cycle = {0: [1], 1: [2], 2: [0]}
print("cycle ->", topo_sort(3, graph_cycle))   # None

# --- DAG 최장 경로 ---
def longest_path_dag(n, graph, weight):
    order = topo_sort(n, graph)
    dp = [0] * n
    for u in order:
        for v in graph[u]:
            if dp[u] + weight[(u, v)] > dp[v]:
                dp[v] = dp[u] + weight[(u, v)]
    return max(dp)

weight = {(0,1):3, (0,2):2, (1,3):4, (2,3):5, (2,4):1}
print("longest path:", longest_path_dag(5, graph, weight))   # 7 (0->2->3)
```
