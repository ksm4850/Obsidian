## 1. 파일 및 디렉터리 관련
| 명령어     | 설명                   | 예시                        |
| ------- | -------------------- | ------------------------- |
| `ls`    | 현재 디렉터리의 파일/폴더 목록 출력 | `ls -al` : 숨김파일 포함 상세 목록  |
| `cd`    | 디렉터리 이동              | `cd /home/user`           |
| `pwd`   | 현재 작업 경로 출력          | `pwd`                     |
| `mkdir` | 새 디렉터리 생성            | `mkdir logs`              |
| `rmdir` | 빈 디렉터리 삭제            | `rmdir old_dir`           |
| `rm`    | 파일 또는 디렉터리 삭제        | `rm -rf temp/`            |
| `cp`    | 파일/디렉터리 복사           | `cp file1.txt /backup/`   |
| `mv`    | 파일/디렉터리 이동 또는 이름 변경  | `mv old.txt new.txt`      |
| `touch` | 빈 파일 생성 또는 수정시간 변경   | `touch new.log`           |
| `find`  | 파일 검색                | `find / -name access.log` |

## 2. 파일 내용 확인
| 명령어 | 설명 | 예시 |
|--------|------|------|
| `cat` | 파일 내용 출력 | `cat config.yaml` |
| `less` | 페이지 단위로 파일 보기 | `less /var/log/syslog` |
| `head` | 파일의 앞부분 출력 | `head -n 20 logfile` |
| `tail` | 파일의 마지막 부분 출력 | `tail -f /var/log/messages` |
| `grep` | 문자열 검색 | `grep "error" app.log` |

## 3. 권한 및 소유권
| 명령어     | 설명           | 예시                          |        |
| ------- | ------------ | --------------------------- | ------ |
| `chmod` | 권한 변경        | `chmod 755 script.sh`       | [[권한]] |
| `chown` | 파일 소유자 변경    | `chown user:group data.txt` |        |
| `umask` | 기본 권한 마스크 설정 | `umask 022`                 |        |

## 4. 프로세스 및 시스템
| 명령어 | 설명 | 예시 |
|--------|------|------|
| `ps` | 현재 프로세스 목록 출력 | `ps aux | grep nginx` |
| `top` | 실시간 시스템 리소스 사용량 | `top` |
| `htop` | 개선된 top 인터페이스 (패키지 필요) | `htop` |
| `kill` | 프로세스 종료 | `kill -9 1234` |
| `df` | 디스크 사용량 확인 | `df -h` |
| `du` | 디렉터리별 용량 확인 | `du -sh /var/log` |
| `free` | 메모리 사용량 확인 | `free -m` |
| `uptime` | 시스템 부팅 후 경과 시간 | `uptime` |

## 5. 네트워크
| 명령어 | 설명 | 예시 |
|--------|------|------|
| `ping` | 네트워크 연결 테스트 | `ping 8.8.8.8` |
| `ifconfig` | 네트워크 인터페이스 정보 (구버전) | `ifconfig eth0` |
| `ip` | 네트워크 설정 및 조회 | `ip addr show` |
| `netstat` | 포트 및 연결 상태 확인 | `netstat -tuln` |
| `ss` | 소켓 상태 확인 (netstat 대체) | `ss -tuln` |
| `curl` | HTTP 요청 테스트 | `curl -I https://example.com` |
| `wget` | 파일 다운로드 | `wget https://example.com/file.zip` |

## 7. 패키지 관리 (배포판별)
| 배포판               | 설치            | 삭제           | 업데이트                        |
| ----------------- | ------------- | ------------ | --------------------------- |
| **Ubuntu/Debian** | `apt install` | `apt remove` | `apt update && apt upgrade` |
| **CentOS/RHEL**   | `yum install` | `yum remove` | `yum update`                |
| **Fedora**        | `dnf install` | `dnf remove` | `dnf update`                |
